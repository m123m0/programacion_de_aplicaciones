# Generación de tickets

Proyecto desarrollado para la asignatura Programación de Aplicaciones Móviles.

La aplicación corresponde a una solución móvil híbrida orientada a la gestión y seguimiento de incidencias, desarrollada con Ionic, Angular y Capacitor. El proyecto considera un enfoque progresivo, incorporando mejoras a partir de la retroalimentación recibida durante el desarrollo del curso.

## Descripción general

La aplicación permite acceder de forma controlada, navegar entre distintas vistas y visualizar el detalle de incidencias registradas. Su diseño prioriza la claridad del flujo, la separación de responsabilidades y una base técnica preparada para procesos de validación y publicación en Android.

## Tecnologías utilizadas

- Ionic
- Angular
- Capacitor
- Android
- Jasmine / Karma
- Cypress
